# SQL Query Generator Bot

A Telegram bot that generates SQL queries from natural language using OpenAI.

## Features
- Multiple database schemas support
- Natural language to SQL conversion
- Query history tracking
- Schema selection

## Setup
1. Install dependencies: `pip install -r requirements.txt`
2. Create `.env` file with your tokens
3. Run: `python bot.py`

## Deployment on Railway
1. Push code to GitHub
2. Connect repository to Railway
3. Add environment variables (BOT_TOKEN, OPENAI_API_KEY)
4. Deploy!
